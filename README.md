# The Law [![Build Status](https://travis-ci.org/mrrusof/the-law.svg?branch=master)](https://travis-ci.org/mrrusof/the-law)

- Make 3.81
- Docker 17.06
- PostgreSQL client 9.6.3
- Flyway 4.2.0
- Bundler 1.16.2
